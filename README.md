# CV Maker

A full-stack web application to track job applications.

**Stack:** React (TypeScript) | FastAPI (Python) | PostgreSQL

## Progress

### Done
* **Database:** Configured PostgreSQL (`cv_maker_db`) with a `jobs` table and `application_status` enum.
* **API:** Initialized a FastAPI project using `uv` as the environment and package manager.
* **API:** Created a basic "Hello World" endpoint (`/`) and confirmed the `uvicorn` server runs.

### Next Steps
1.  **API:** Connect FastAPI to the PostgreSQL database.
2.  **API:** Build CRUD (Create, Read, Update, Delete) endpoints for the `jobs` table.
3.  **Front End:** Create the React/TypeScript application.
4.  **Integration:** Connect the React app to the FastAPI API.
5.  **UI:** Build the user interface to display and manage job applications.
